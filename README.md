# filtr

> MIDI transformation engine with 20 transforms

## Controls

- **E1** — select transform
- **E2** — select parameter
- **E3** — adjust parameter value
- **K2** — toggle selected transform on/off
- **K3** — reset selected parameter to default

## Grid

8×16 layout: row 1 = transforms 1–16; row 2 = transforms 17–20; rows 3–5 = parameter value bars (with row 8 showing on/off and param selection).

## Requirements

- norns
- grid (16x8)
- MIDI device (input + output)

## Install

```
;install https://github.com/jamminstein/filtr
```
